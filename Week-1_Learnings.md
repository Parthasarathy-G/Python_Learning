# Week 1 Learnings

## Day 1
- Studied Python fundamentals: origin, common use cases, and development tools.
- Practiced basic programs (arithmetic operations and function calls).
- Learned LEGB rule (Local, Enclosing, Global, Built-in) for variable scope.
- Understood indentation-based syntax and block structure in Python.

## Day 2
- Practiced conditional statements, input handling, loops, and string manipulation.
- Worked on basic real-world style examples inspired by apps like Uber, Zomato, and Amazon.

## Day 3
- Studied OOP concepts: classes, objects, attributes, and methods.
- Practiced constructors and instance variables.
- Learned inheritance and polymorphism for code reuse.
- Explored encapsulation and access modifiers (public/private/protected).

## Day 4
- Studied exception handling using `try`, `except`, and `finally`.
- Learned built-in exceptions and custom exception ideas.
- Practiced file handling: reading, writing, and appending.
- Explored text, CSV, and JSON as persistence formats.

## Day 5
- Studied inheritance and its types.
- Created constructor and class-object examples.

## Day 6
- Studied polymorphism, access specifiers, and access methods.

## Day 7
- Practiced encapsulation with class, instance, and method-level examples.
- Continued file management practice for read/write/append workflows.
